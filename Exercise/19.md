## 1. 問題タイトル

19 本目：CylinderGeometry で円柱

## 2. 学習目標

この演習を通じて、以下の技術を習得します。

- `CylinderGeometry`（円柱）の作成
- 上面と底面の半径を変えた円錐台の作成

## 3. 新しい概念の解説

**CylinderGeometry（円柱ジオメトリ）**
円柱を作成します。
`radiusTop`（上面の半径）と `radiusBottom`（底面の半径）を個別に指定できるため、円錐台（コップ型）や円錐（コーン型）を作ることも可能です。

```javascript
// 上面半径5, 底面半径5, 高さ20, 分割数32
const cylinder = new THREE.CylinderGeometry(5, 5, 20, 32);

// 上面半径を0にすると円錐（コーン）になります
const cone = new THREE.CylinderGeometry(0, 5, 20, 32);
```

## 4. 課題の説明

円柱（シリンダー）を作成して表示してください。
また、上面の半径を小さくして、コップやバケツのような形にしてみてください。

## 5. 必須要件

以下の機能を実装してください。

- **円錐台の作成**:
  `new THREE.CylinderGeometry(0.5, 1, 2, 32)` を作成します。
  引数は `(radiusTop, radiusBottom, height, radialSegments)` です。
  上面半径(0.5) < 底面半径(1) にすることで、コップのような形にします。

- **表示**:
  マテリアルを適用し、回転させて形状を確認します。立体感が分かりにくい場合は `wireframe: true` やライトを使用してください。

## 6. ヒント

- **ヒント 1（円錐台）**:
  上面（radiusTop）と底面（radiusBottom）の半径を変えることで、バケツやプリンのような形（円錐台）を作れます。
- **ヒント 2（円錐）**:
  radiusTop を `0` に設定すると、先端が尖った「円錐（Cone）」になります。
- **ヒント 3（多角柱）**:
  `radialSegments`（側面の分割数）を `3` にすると三角柱、`4` にすると四角柱、`6` にすると六角柱になります。円柱以外の柱を作るときにも使えます。
- **ヒント 4（蓋の有無）**:
  `openEnded` 引数を `true` にすると、上と下の蓋がなくなり、トイレットペーパーの芯のような筒になります。
- **ヒント 5（高さの基準）**:
  高さ（height）は Y 軸方向の長さです。中心から上下に伸びます。

## 8. 模範解答

```javascript
import * as THREE from "three";

const scene = new THREE.Scene();
const camera = new THREE.PerspectiveCamera(
  75,
  window.innerWidth / window.innerHeight,
  0.1,
  1000
);
const renderer = new THREE.WebGLRenderer();
renderer.setSize(window.innerWidth, window.innerHeight);
document.body.appendChild(renderer.domElement);

// 1. CylinderGeometryの作成
// 上面0.5, 底面1, 高さ2, 分割数32
const geometry = new THREE.CylinderGeometry(0.5, 1, 2, 32);
const material = new THREE.MeshBasicMaterial({
  color: 0xffff00,
  wireframe: true,
});
const cylinder = new THREE.Mesh(geometry, material);
scene.add(cylinder);

camera.position.z = 5;

function animate() {
  requestAnimationFrame(animate);

  cylinder.rotation.x += 0.01;
  cylinder.rotation.z += 0.01;

  renderer.render(scene, camera);
}
animate();
```
