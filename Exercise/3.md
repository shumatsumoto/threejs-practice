## 1. 問題タイトル

3 本目：SphereGeometry で球体を作成

## 2. 学習目標

この演習を通じて、以下の技術を習得します。

- 異なるジオメトリ（球体）の使用方法
- マテリアルのオプション設定（ワイヤーフレーム表示）
- ポリゴン（多角形）による曲面表現の理解

## 3. 新しい概念の解説

**SphereGeometry（球体ジオメトリ）**
球体を作成するための形状データです。
`SphereGeometry(radius, widthSegments, heightSegments)` のように指定します。
コンピュータグラフィックスでは完全な球体は作れないため、細かい三角形の集まりで近似します。分割数（Segments）が多いほど滑らかになりますが、処理負荷も上がります。

```javascript
// 半径1, 横分割数32, 縦分割数16の球体
const geometry = new THREE.SphereGeometry(1, 32, 16);
```

**Wireframe（ワイヤーフレーム）**
ポリゴンの「網目」だけを表示する描画モードです。物体の構造を確認するのに便利です。
マテリアルのオプションで `wireframe: true` に設定すると有効になります。

```javascript
const material = new THREE.MeshBasicMaterial({
  color: 0x00ff00,
  wireframe: true // ワイヤーフレームを有効化
});
```

## 4. 課題の説明

立方体の代わりに、青色の球体（スフィア）を表示してください。
さらに、球体がどのような構造でできているか確認するために、ワイヤーフレーム（網目）モードで表示してください。

## 5. 必須要件

以下の機能を実装してください。

- **SphereGeometryの作成**:
  `new THREE.SphereGeometry(1, 32, 16)` を作成します。
  引数は `(radius, widthSegments, heightSegments)` です。

- **ワイヤーフレームマテリアル**:
  `new THREE.MeshBasicMaterial({ color: 0x0000ff, wireframe: true })` を作成します。
  青色（`0x0000ff`）を指定し、`wireframe: true` オプションを有効にします。

- **表示**:
  メッシュを作成してシーンに追加し、青い網目の球体が表示されることを確認します。

## 6. ヒント

- **ヒント 1（ジオメトリの引数）**:
  `new THREE.SphereGeometry(1, 32, 16)` の引数は順に「半径」「経度方向の分割数」「緯度方向の分割数」です。
  分割数を減らすと（例: 8, 8）、カクカクした宝石のような見た目になります。
- **ヒント 2（ワイヤーフレーム）**:
  マテリアル作成時に `{ wireframe: true }` を指定すると、面を塗りつぶさずに線だけで描画されます。
  ポリゴンの構造（三角形の集まり）を目で見て理解するのに非常に役立ちます。
- **ヒント 3（色の指定）**:
  色は 16 進数 `0x0000ff`（青）で指定します。`Color` クラスの仕様で、CSS のように `'blue'` という文字列でも指定可能です。
- **ヒント 4（メッシュの作成）**:
  ジオメトリだけを変えても、前回の `BoxGeometry` と同じ手順（Mesh 作成 -> scene.add）で表示できます。これが Three.js の優れた設計です。
- **ヒント 5（BasicMaterial）**:
  今回も光源を設定していないため、`MeshBasicMaterial` を使うのが無難です。`MeshStandardMaterial` などはライトがないと真っ黒になります。

## 8. 模範解答

```javascript
import * as THREE from "three";

const scene = new THREE.Scene();
const camera = new THREE.PerspectiveCamera(
  75,
  window.innerWidth / window.innerHeight,
  0.1,
  1000
);
const renderer = new THREE.WebGLRenderer();
renderer.setSize(window.innerWidth, window.innerHeight);
document.body.appendChild(renderer.domElement);

// 1. 球体ジオメトリを作成
// 半径1, 横分割数32, 縦分割数16
const geometry = new THREE.SphereGeometry(1, 32, 16);

// 2. 青色のワイヤーフレームマテリアルを作成
const material = new THREE.MeshBasicMaterial({
  color: 0x0000ff,
  wireframe: true,
});

// 3. メッシュを作成してシーンに追加
const sphere = new THREE.Mesh(geometry, material);
scene.add(sphere);

camera.position.z = 5;
renderer.render(scene, camera);
```
