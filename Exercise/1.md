## 1. 問題タイトル

Three.js の基本：シーン、カメラ、レンダラーの初期設定

## 2. 課題の概要

Three.js を使って、何もない真っ黒な 3D 空間をブラウザ画面いっぱいに表示してください。
これはすべての Three.js プロジェクトの出発点となる最も重要なステップです。
「シーン（舞台）」、「カメラ（視点）」、「レンダラー（映写機）」という 3 つの基本要素の関係性を理解することが目標です。

## 3. 新しい概念の解説

**Scene（シーン）**
3D オブジェクトを配置する「舞台」です。すべてのオブジェクト、ライト、カメラはシーンに追加されます。

使い方:

```javascript
const scene = new THREE.Scene();
scene.background = new THREE.Color(0x000000); // 背景色を黒に設定
scene.add(mesh); // メッシュをシーンに追加
```

**Camera（カメラ）**
3D 空間をどの視点から見るかを決定します。`PerspectiveCamera`は人間の目のような遠近感のあるカメラです。

使い方:

```javascript
// PerspectiveCamera(視野角, アスペクト比, 近クリップ面, 遠クリップ面)
const camera = new THREE.PerspectiveCamera(
  75,
  window.innerWidth / window.innerHeight,
  0.1,
  1000
);
camera.position.z = 5; // カメラを後ろに移動
```

**Renderer（レンダラー）**
シーンとカメラの情報を元に、実際に画面に描画する役割を持ちます。WebGL を使って GPU で高速に描画します。

使い方:

```javascript
const renderer = new THREE.WebGLRenderer();
renderer.setSize(window.innerWidth, window.innerHeight); // サイズ設定
document.body.appendChild(renderer.domElement); // DOMに追加
renderer.render(scene, camera); // 描画実行
```

## 4. 課題の説明

Three.js を使って、何もない真っ黒な 3D 空間をブラウザ画面いっぱいに表示してください。
これはすべての Three.js プロジェクトの出発点となる最も重要なステップです。

## 5. 必須要件

以下の機能を実装してください。

- **STEP 1: シーンの作成**
  `THREE.Scene` クラスをインスタンス化し、変数 `scene` に代入してください。

- **STEP 2: カメラの作成**
  `THREE.PerspectiveCamera` クラスをインスタンス化し、変数 `camera` に代入してください。
  引数は `(75, window.innerWidth / window.innerHeight, 0.1, 1000)` とします。

- **STEP 3: レンダラーの作成と設定**
  `THREE.WebGLRenderer` クラスをインスタンス化し、変数 `renderer` に代入してください。
  `renderer.setSize()` でウィンドウサイズを設定します。

- **STEP 4: DOM への追加**
  `document.body.appendChild(renderer.domElement)` を実行して Canvas をページに追加します。

- **STEP 5: 描画の実行**
  `renderer.render(scene, camera)` を呼び出して描画します。

## 6. ヒント

- **ヒント 1（3 大要素）**:
  Three.js で何かを表示するには、必ず `Scene`（場所）、`Camera`（目）、`Renderer`（描画機能）の 3 つが必要です。これらはセットで覚えると良いでしょう。
- **ヒント 2（カメラの引数）**:
  `PerspectiveCamera(fov, aspect, near, far)` です。特に `aspect`（アスペクト比）は画面の縦横比に合わせないと、映像が歪んでしまいます。
- **ヒント 3（レンダラーの役割）**:
  レンダラーは、シーンとカメラの情報を使って、実際にブラウザの `<canvas>` タグに絵を描き出す役割を持っています。`setSize` でキャンバスの大きさを決め、`render` で「今この瞬間の絵」を描きます。

## 8. 模範解答

```javascript
import * as THREE from "three";

// STEP 1: シーンを作成
// オブジェクトやライトを配置するコンテナ
const scene = new THREE.Scene();

// STEP 2: カメラを作成
// PerspectiveCamera(視野角, アスペクト比, 近クリップ面, 遠クリップ面)
const camera = new THREE.PerspectiveCamera(
  75,
  window.innerWidth / window.innerHeight,
  0.1,
  1000
);

// カメラ位置の調整（今回は何もないので必須ではないが、習慣として）
camera.position.z = 5;

// STEP 3: レンダラーを作成
const renderer = new THREE.WebGLRenderer();

// レンダラーのサイズをウィンドウサイズに合わせる
renderer.setSize(window.innerWidth, window.innerHeight);

// STEP 4: レンダラーをDOMに追加
// これによりHTMLに<canvas>要素が挿入される
document.body.appendChild(renderer.domElement);

// STEP 5: 描画を実行
// シーンをカメラの視点からレンダリングする
renderer.render(scene, camera);
```
