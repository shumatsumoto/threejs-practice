## 1. 問題タイトル

11 本目：複数のオブジェクトを配置

## 2. 学習目標

この演習を通じて、以下の技術を習得します。

- 複数のメッシュの作成と管理
- オブジェクトごとの位置調整
- シーン構成の基礎

## 3. 新しい概念の解説

**Multiple Objects（複数のオブジェクト）**
Three.js のシーンには、メモリが許す限りいくつでもオブジェクトを追加できます。
それぞれ独立した `mesh` 変数として作成し、個別に `position` や `rotation` を制御できます。

```javascript
const mesh1 = new THREE.Mesh(geo, mat1);
const mesh2 = new THREE.Mesh(geo, mat2);
scene.add(mesh1);
scene.add(mesh2);
```

**Coordinate System（座標系）**
複数のオブジェクトを配置する際は、位置（X, Y, Z）が重ならないように注意が必要です。
`position.x = -2`（左）と `position.x = 2`（右）のように配置することで並べて表示できます。

```javascript
mesh1.position.x = -2;
mesh2.position.x = 2;
```

## 4. 課題の説明

シーンに「緑のキューブ」と「青い球体」の 2 つを同時に表示してください。
それらが重ならないように、位置をずらして配置してください。

## 5. 必須要件

以下の機能を実装してください。

- **2つのメッシュ作成**:
  1. `BoxGeometry` と `MeshBasicMaterial`（緑）を使ったキューブを作成します。
  2. `SphereGeometry` と `MeshBasicMaterial`（青・ワイヤーフレーム）を使った球体を作成します。

- **配置**:
  キューブを左側（`position.x = -2`）に、球体を右側（`position.x = 2`）に配置します。

- **シーンへの追加**:
  両方のメッシュを `scene.add()` します。

- **アニメーション**:
  `animate` ループ内で、両方のオブジェクトをそれぞれ異なる軸（例：キューブはX軸、球体はY軸）で回転させます。

## 6. ヒント

- **ヒント 1（独立性）**:
  `mesh1` と `mesh2` は完全に独立したオブジェクトです。それぞれ違うジオメトリ、マテリアル、位置を持つことができます。
- **ヒント 2（配置座標）**:
  画面中央が X=0 です。`x = -2` は左側、`x = 2` は右側に配置されます。
  重ならないように十分な距離を空けるのがポイントです。
- **ヒント 3（別々の変数）**:
  `const cube = ...` と `const sphere = ...` のように変数名を分けることで、後から個別に操作（回転など）できます。
- **ヒント 4（シーンへの追加）**:
  オブジェクトを 2 つ作ったら、`scene.add(cube); scene.add(sphere);` のように**両方とも**シーンに追加する必要があります。片方忘れると表示されません。
- **ヒント 5（ループ内での操作）**:
  `animate` 関数の中で `cube.rotation.x += ...` と `sphere.rotation.y += ...` のように書けば、それぞれ独自のアニメーションをさせることができます。

## 7. スターターコード

```javascript
import * as THREE from "three";

// ...基本セットアップ...

// 1. 緑のキューブを作成

// 2. 青い球体を作成
```

## 8. 模範解答

```javascript
import * as THREE from "three";

const scene = new THREE.Scene();
const camera = new THREE.PerspectiveCamera(
  75,
  window.innerWidth / window.innerHeight,
  0.1,
  1000
);
const renderer = new THREE.WebGLRenderer();
renderer.setSize(window.innerWidth, window.innerHeight);
document.body.appendChild(renderer.domElement);

// 1. 緑のキューブ
const boxGeo = new THREE.BoxGeometry(1, 1, 1);
const boxMat = new THREE.MeshBasicMaterial({ color: 0x00ff00 });
const cube = new THREE.Mesh(boxGeo, boxMat);
cube.position.x = -2; // 左へ
scene.add(cube);

// 2. 青い球体
const sphereGeo = new THREE.SphereGeometry(0.7, 32, 16);
const sphereMat = new THREE.MeshBasicMaterial({
  color: 0x0000ff,
  wireframe: true,
});
const sphere = new THREE.Mesh(sphereGeo, sphereMat);
sphere.position.x = 2; // 右へ
scene.add(sphere);

camera.position.z = 5;

function animate() {
  requestAnimationFrame(animate);

  // 両方を回転
  cube.rotation.x += 0.01;
  cube.rotation.y += 0.01;

  sphere.rotation.x -= 0.01;
  sphere.rotation.y -= 0.01;

  renderer.render(scene, camera);
}

animate();
```
