## 1. 問題タイトル

2 本目：BoxGeometry で立方体を表示

## 2. 学習目標

この演習を通じて、以下の技術を習得します。

- 3D オブジェクトの構成要素である「ジオメトリ（形状）」と「マテリアル（材質）」の概念
- それらを組み合わせて「メッシュ（物体）」を作成する方法
- シーンへのオブジェクトの追加方法

## 3. 新しい概念の解説

**Geometry（ジオメトリ）**
3D オブジェクトの「形状」を定義します。頂点（Vertex）や面（Face）の情報を持っています。
今回は `BoxGeometry`（直方体）を使用します。

```javascript
// 幅1, 高さ1, 奥行き1の直方体ジオメトリ
const geometry = new THREE.BoxGeometry(1, 1, 1);
```

**Material（マテリアル）**
3D オブジェクトの「見た目」や「材質」を定義します。色、光の反射、透明度などを設定します。
今回は `MeshBasicMaterial`（光の影響を受けない単色素材）を使用します。

```javascript
// 赤色(0xff0000)のマテリアル
const material = new THREE.MeshBasicMaterial({ color: 0xff0000 });
```

**Mesh（メッシュ）**
ジオメトリとマテリアルを組み合わせた、実際にシーンに配置される「物体」です。

```javascript
// 形と材質を組み合わせてメッシュ(物体)を作成
const mesh = new THREE.Mesh(geometry, material);
```

**Scene.add**
作成したメッシュをシーンに追加（登録）するメソッドです。これをしないと画面には何も表示されません。

```javascript
scene.add(mesh);
```

## 4. 課題の説明

演習 1 で作成したシーンに、緑色の立方体（キューブ）を一つ表示してください。
カメラの位置を調整して、キューブが正しく見えるようにしてください。

## 5. 必須要件

以下の機能を実装してください。

- **ジオメトリ作成**:
  `new THREE.BoxGeometry(1, 1, 1)` を作成します。

- **マテリアル作成**:
  `new THREE.MeshBasicMaterial({ color: 0x00ff00 })`（緑色）を作成します。

- **メッシュ作成**:
  上記のジオメトリとマテリアルを引数にして `new THREE.Mesh(geometry, material)` を作成します。

- **シーンへの追加**:
  `scene.add(mesh)` を実行し、メッシュをシーンに登録します。

- **カメラの配置**:
  初期状態ではカメラとオブジェクトが重なっているので、`camera.position.z = 5` などに設定して手前に引きます。
  緑色の正方形（正面から見た立方体）が表示されれば成功です。

## 6. ヒント

- **ヒント 1（3D オブジェクトの構成）**:
  Three.js では「形（Geometry）」と「見た目（Material）」を別々に作り、それを「Mesh」という箱に入れて一つにします。これらをセットにしないと表示されません。
- **ヒント 2（BoxGeometry）**:
  `new THREE.BoxGeometry(1, 1, 1)` で幅・高さ・奥行きがすべて 1 の立方体が作れます。引数を変えると細長い棒や板も作れます。
- **ヒント 3（MeshBasicMaterial）**:
  `new THREE.MeshBasicMaterial({ color: 0x00ff00 })` で緑色の素材が作れます。`BasicMaterial` はライトがなくても明るく見えるため、初心者のテスト用に最適です。
- **ヒント 4（シーンへの追加）**:
  作成したメッシュは `scene.add(mesh)` しないと画面に登場しません。これは「撮影セットに俳優を立たせる」ような意味です。
- **ヒント 5（カメラ位置）**:
  初期状態ではカメラもキューブも原点(0,0,0)に重なっています。
  `camera.position.z = 5` のようにカメラを Z 軸プラス方向（手前）に引かないと、オブジェクトの中にカメラが埋まってしまい何も見えません。

## 8. 模範解答

```javascript
import * as THREE from "three";

const scene = new THREE.Scene();
const camera = new THREE.PerspectiveCamera(
  75,
  window.innerWidth / window.innerHeight,
  0.1,
  1000
);
const renderer = new THREE.WebGLRenderer();
renderer.setSize(window.innerWidth, window.innerHeight);
document.body.appendChild(renderer.domElement);

// 1. ジオメトリ（形状）を作成
// BoxGeometry(幅, 高さ, 奥行き)
const geometry = new THREE.BoxGeometry(1, 1, 1);

// 2. マテリアル（材質）を作成
// 色は16進数で指定 (緑: 0x00ff00)
const material = new THREE.MeshBasicMaterial({ color: 0x00ff00 });

// 3. メッシュ（物体）を作成
const cube = new THREE.Mesh(geometry, material);

// 4. シーンに追加
scene.add(cube);

// 5. カメラの位置を調整
// 手前（Zプラス方向）に5移動
camera.position.z = 5;

renderer.render(scene, camera);
```
