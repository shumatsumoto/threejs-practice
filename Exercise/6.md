## 1. 問題タイトル

6 本目：オブジェクトの回転

## 2. 学習目標

この演習を通じて、以下の技術を習得します。

- 3D 空間での回転（Rotation）の操作
- ラジアン（弧度法）と度数法の違い
- オイラー角（Euler Angles）の概念

## 3. 新しい概念の解説

**Rotation（回転）**
オブジェクトの傾きを定義するプロパティです。
`cube.rotation.x` のように軸ごとに指定します。
単位は「度（Degree）」ではなく「ラジアン（Radian）」を使用するのが基本です。

```javascript
// X軸周りに回転を設定
cube.rotation.x = 0.5;
```

**Radian（ラジアン）**
角度の単位です。
円の半径と同じ長さの弧に対する角度が 1 ラジアンです。
- 180度 = π (約3.14) ラジアン
- 360度 = 2π ラジアン
Three.js では `Math.PI` を使って計算します（例: 45度 = `Math.PI / 4`）。

```javascript
// 90度（π/2）回転
cube.rotation.y = Math.PI / 2;

// 度数法からの変換公式
const degree = 45;
const radian = degree * (Math.PI / 180);
cube.rotation.z = radian;
```

## 4. 課題の説明

キューブを少し傾けて、立体的に見えるようにしてください。
X 軸周りと Y 軸周りに 45 度ずつ回転させます。

## 5. 必須要件

以下の機能を実装してください。

- **回転の設定**:
  キューブの `rotation` プロパティを操作します。

- **ラジアン指定**:
  角度は必ずラジアンで指定します。45度は `Math.PI / 4` です。

- **回転の適用**:
  X 軸周りに 45度、Y 軸周りにも 45度回転させます。
  `cube.rotation.x = Math.PI / 4;`
  `cube.rotation.y = Math.PI / 4;`

- **確認**:
  キューブが傾いて、立体的な角が見えるようになることを確認します。

## 6. ヒント

- **ヒント 1（ラジアンの重要性）**:
  3D プログラミングや数学関数（sin, cos）では、角度は常に「ラジアン」を使います。
  「度数法（Degree）」で指定したい場合は変換が必要です。
- **ヒント 2（変換公式）**:
  「180 度 = Math.PI」と覚えれば計算できます。
  45 度は 180 度の 1/4 なので、`Math.PI / 4` です。
- **ヒント 3（回転軸）**:
  `rotation.x` は「焼き鳥の串」のように X 軸を中心にして回るイメージです。
  手前に倒れたり起き上がったりする回転になります。
- **ヒント 4（回転順序）**:
  Three.js ではデフォルトで「X -> Y -> Z」の順に回転が適用されます（Euler 角）。
  順序によって最終的な姿勢が変わることがありますが、今回は気にしなくて大丈夫です。
- **ヒント 5（値の指定）**:
  `cube.rotation.x = Math.PI / 4;`
  `cube.rotation.y = Math.PI / 4;`
  のように代入してください。正の値だと反時計回り（軸のプラス方向から見て）になります。

## 7. スターターコード

```javascript
import * as THREE from "three";

const scene = new THREE.Scene();
const camera = new THREE.PerspectiveCamera(
  75,
  window.innerWidth / window.innerHeight,
  0.1,
  1000
);
const renderer = new THREE.WebGLRenderer();
renderer.setSize(window.innerWidth, window.innerHeight);
document.body.appendChild(renderer.domElement);

const geometry = new THREE.BoxGeometry(1, 1, 1);
const material = new THREE.MeshBasicMaterial({ color: 0x00ff00 });
const cube = new THREE.Mesh(geometry, material);
scene.add(cube);

camera.position.z = 5;

renderer.render(scene, camera);
```

## 8. 模範解答

```javascript
import * as THREE from "three";

const scene = new THREE.Scene();
const camera = new THREE.PerspectiveCamera(
  75,
  window.innerWidth / window.innerHeight,
  0.1,
  1000
);
const renderer = new THREE.WebGLRenderer();
renderer.setSize(window.innerWidth, window.innerHeight);
document.body.appendChild(renderer.domElement);

const geometry = new THREE.BoxGeometry(1, 1, 1);
const material = new THREE.MeshBasicMaterial({ color: 0x00ff00 });
const cube = new THREE.Mesh(geometry, material);
scene.add(cube);

camera.position.z = 5;

// 回転の変更（ラジアン指定）
// X軸周りに45度
cube.rotation.x = Math.PI / 4;
// Y軸周りに45度
cube.rotation.y = Math.PI / 4;

renderer.render(scene, camera);
```
