## 1. 問題タイトル

81 本目：ドラッグ＆ドロップ操作（DragControls）

## 2. 学習目標

この演習を通じて、以下の技術を習得します。

- `DragControls` の使用方法
- マウスドラッグによるオブジェクトの移動
- ドラッグ開始・終了時のイベント処理（色変更など）

## 3. 新しい概念の解説

**DragControls（ドラッグ操作）**
マウスやタッチ操作でオブジェクトを直接掴んで移動させるためのクラスです。
Three.js の標準ライブラリには含まれていませんが、`examples/jsm/controls/DragControls` からインポートして使用できます。
Raycaster を内部で使って当たり判定を行っており、簡単にインタラクティブな操作を実装できます。

```javascript
import { DragControls } from 'three/examples/jsm/controls/DragControls';

const controls = new DragControls(objects, camera, renderer.domElement);

controls.addEventListener('dragstart', function (event) {
  // ドラッグ中は OrbitControls などのカメラ操作と干渉しないように無効化する
  orbitControls.enabled = false;
  event.object.material.emissive.set(0xaaaaaa); // 色を変えて掴んでいることを表現
});

controls.addEventListener('dragend', function (event) {
  orbitControls.enabled = true;
  event.object.material.emissive.set(0x000000);
});
```

## 4. 課題の説明

3D 空間にある複数のオブジェクトを、マウスドラッグで自由に移動できるようにしてください。
`DragControls` を使うと、レイキャスターの計算を自前で書かなくても、簡単にドラッグ操作を実装できます。
ドラッグ中はオブジェクトの色を変えて、掴んでいることが分かるようにします。

## 5. 必須要件

以下の機能を実装してください。

- **DragControls のセットアップ**:
  `examples/jsm/controls/DragControls` をインポートし、`new DragControls(objects, camera, renderer.domElement)` でインスタンスを作成します。第一引数の `objects` は、ドラッグ対象にしたいメッシュを入れた配列です。

- **ドラッグ開始時の処理 (`dragstart`)**:
  イベントリスナーを追加し、以下の処理を行います。
  1. `orbitControls.enabled = false` にして、ドラッグ中にカメラが動かないようにする。
  2. `event.object.material.emissive.set(0xaaaaaa)` などで、掴んでいるオブジェクトを光らせる。

- **ドラッグ終了時の処理 (`dragend`)**:
  1. `orbitControls.enabled = true` に戻す。
  2. `event.object.material.emissive.set(0x000000)` で色を元に戻す。

- **対象オブジェクト**:
  箱や球体など、適当なメッシュを複数作成してシーンに追加し、同時に `objects.push(mesh)` で配列にも追加しておきます。

## 6. ヒント

- **ヒント 1（インポート）**:
  `DragControls` はコアカテゴリではなく `examples/jsm/controls/DragControls` にあります。
- **ヒント 2（操作の競合回避）**:
  `OrbitControls` と `DragControls` を同時に使うと、ドラッグ時にカメラも動いてしまい操作しづらいです。
  `dragstart` で `orbitControls.enabled = false`、`dragend` で `true` に切り替えるのが定石です。
- **ヒント 3（イベントリスナー）**:
  `controls.addEventListener('dragstart', (event) => { ... })`
  `event.object` で現在掴んでいるメッシュにアクセスできます。これを使って色を変えたりできます。
- **ヒント 4（対象オブジェクト）**:
  コンストラクタの第一引数に渡す配列 `objects` は参照渡しです。
  シーンに新しい物体を追加した場合、この配列にも `push` しないとドラッグ対象になりません。
- **ヒント 5（再帰的処理）**:
  `transformGroup` オプションを使うと、個々のメッシュではなくグループ全体をドラッグ可能にするなどの高度な設定もできます。

## 8. 模範解答

```javascript
import * as THREE from "three";
import { DragControls } from "three/examples/jsm/controls/DragControls";
import { OrbitControls } from "three/examples/jsm/controls/OrbitControls";

const scene = new THREE.Scene();
scene.background = new THREE.Color(0xf0f0f0);

const camera = new THREE.PerspectiveCamera(
  75,
  window.innerWidth / window.innerHeight,
  0.1,
  1000
);
const renderer = new THREE.WebGLRenderer({ antialias: true });
renderer.setSize(window.innerWidth, window.innerHeight);
document.body.appendChild(renderer.domElement);

// ライト
scene.add(new THREE.AmbientLight(0xaaaaaa));
const light = new THREE.SpotLight(0xffffff);
light.position.set(0, 25, 50);
light.angle = Math.PI / 5;
light.penumbra = 0.2;
light.decay = 2;
light.distance = 200;
light.castShadow = true;
scene.add(light);

// オブジェクト生成
const objects = [];
const geometry = new THREE.BoxGeometry(1, 1, 1);

for (let i = 0; i < 5; i++) {
  const material = new THREE.MeshLambertMaterial({
    color: Math.random() * 0xffffff,
  });
  const mesh = new THREE.Mesh(geometry, material);

  mesh.position.x = (Math.random() - 0.5) * 10;
  mesh.position.y = (Math.random() - 0.5) * 6;
  mesh.position.z = (Math.random() - 0.5) * 4;

  mesh.castShadow = true;
  mesh.receiveShadow = true;

  scene.add(mesh);
  objects.push(mesh);
}

// OrbitControls（カメラ操作用）
const orbitControls = new OrbitControls(camera, renderer.domElement);

// DragControls（オブジェクト操作用）
const dragControls = new DragControls(objects, camera, renderer.domElement);

// イベントリスナー
dragControls.addEventListener("dragstart", function (event) {
  // ドラッグ中はカメラ操作を無効化
  orbitControls.enabled = false;
  // 色を変えて掴んでいることを強調
  event.object.material.emissive.set(0xaaaaaa);
});

dragControls.addEventListener("dragend", function (event) {
  // ドラッグ終了でカメラ操作を有効化
  orbitControls.enabled = true;
  // 色を戻す
  event.object.material.emissive.set(0x000000);
});

camera.position.z = 10;

function animate() {
  requestAnimationFrame(animate);
  renderer.render(scene, camera);
}
animate();
```
