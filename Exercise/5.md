## 1. 問題タイトル

5 本目：オブジェクトの位置を変更

## 2. 学習目標

この演習を通じて、以下の技術を習得します。

- 3 次元座標系（X, Y, Z）の理解
- `position` プロパティ（Vector3）の操作方法
- `set` メソッドによる一括設定

## 3. 新しい概念の解説

**Position（位置座標）**
3D オブジェクトが空間のどこにあるかを示すプロパティです。`Vector3` クラス（3次元ベクトル）で管理されます。
- X軸: 左右（+が右）
- Y軸: 上下（+が上）
- Z軸: 前後（+が手前）

```javascript
cube.position.x = 5; // X座標を5に設定
```

**Vector3.set()**
X, Y, Z の3つの値を一度に設定するメソッドです。
`cube.position.set(x, y, z)` のように使用します。コードが簡潔になります。

```javascript
// X=2, Y=1.5, Z=0 に設定
cube.position.set(2, 1.5, 0);
```

## 4. 課題の説明

キューブを画面の「右上」の方へ移動させてください。
3D 空間における座標の感覚を掴むための練習です。

## 5. 必須要件

以下の機能を実装してください。

- **位置の変更**:
  作成したキューブの `position` プロパティを操作します。
  X 軸方向に `2`（右へ）、Y 軸方向に `1.5`（上へ）移動させます。
  Z 軸は `0` のままにします。

- **実装方法**:
  `cube.position.x = 2;` のように個別に設定するか、
  `cube.position.set(2, 1.5, 0);` で一括設定します。

- **確認**:
  キューブが中央から右上にずれて表示されることを確認します。

## 6. ヒント

- **ヒント 1（座標系のルール）**:
  Three.js は「右手座標系」です。画面に向かって
  - 右が X プラス
  - 上が Y プラス
  - 手前が Z プラス
  です。
- **ヒント 2（個別設定）**:
  `cube.position.x = 2;`
  `cube.position.y = 1.5;`
  のように、軸ごとに個別に数値を代入できます。
- **ヒント 3（一括設定）**:
  `cube.position.set(2, 1.5, 0);`
  を使うと、一行ですっきりと記述できます。
- **ヒント 4（相対位置）**:
  `position` は親要素（ここでは Scene = 世界全体）の中心からの距離です。
- **ヒント 5（カメラとの距離）**:
  カメラが `z = 5` にあり、物体が `z = 0` なら距離は 5 です。
  物体を `z = 2` にすると、カメラとの距離は 3 になり、より大きく（近く）見えます。

## 7. スターターコード

```javascript
import * as THREE from "three";

const scene = new THREE.Scene();
const camera = new THREE.PerspectiveCamera(
  75,
  window.innerWidth / window.innerHeight,
  0.1,
  1000
);
const renderer = new THREE.WebGLRenderer();
renderer.setSize(window.innerWidth, window.innerHeight);
document.body.appendChild(renderer.domElement);

const geometry = new THREE.BoxGeometry(1, 1, 1);
const material = new THREE.MeshBasicMaterial({ color: 0x00ff00 });
const cube = new THREE.Mesh(geometry, material);
scene.add(cube);

camera.position.z = 5;

// --- ここで位置を変更してください ---

// --- ここまで ---

renderer.render(scene, camera);
```

## 8. 模範解答

```javascript
import * as THREE from "three";

const scene = new THREE.Scene();
const camera = new THREE.PerspectiveCamera(
  75,
  window.innerWidth / window.innerHeight,
  0.1,
  1000
);
const renderer = new THREE.WebGLRenderer();
renderer.setSize(window.innerWidth, window.innerHeight);
document.body.appendChild(renderer.domElement);

const geometry = new THREE.BoxGeometry(1, 1, 1);
const material = new THREE.MeshBasicMaterial({ color: 0x00ff00 });
const cube = new THREE.Mesh(geometry, material);
scene.add(cube);

camera.position.z = 5;

// 位置の変更
// X軸プラス方向（右）へ2
cube.position.x = 2;
// Y軸プラス方向（上）へ1.5
cube.position.y = 1.5;

// 別解: setメソッド
// cube.position.set(2, 1.5, 0);

renderer.render(scene, camera);
```
